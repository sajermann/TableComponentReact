# Table Component React

[Codesandbox](https://codesandbox.io/p/github/sajermann/TableComponentReact/develop)

## Português

Esse projeto foi criado para demonstrar como eu implemento um componente de tabela, fiz de duas maneiras, uma utilizando o padrão tradicional nomeado por Table, e uma utilizando o padrão de composição nomeado por TableMega.O intuito é deixar o mais fácil possível para servir de consulta posteriores, sendo possível copiar e colocar facilmente.

## English

This project was created to demonstrate how I implement a table component. I did it in two ways: one using the traditional pattern named Table, and another using the composition pattern named TableMega. The goal is to make it as easy as possible to serve as a reference for future use, allowing it to be easily copied and pasted.
